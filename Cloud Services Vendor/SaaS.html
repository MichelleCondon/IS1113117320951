Software as a Service

Software as a service is a way of delivering centrally hosted applications over the Internet, as a service. SaaS applications are sometimes called web-based software, on-demand software, or hosted software. Whatever the name, SaaS applications run on a SaaS provider's servers. Instead of installing complicated and time consuming software it can be accessed via the internet. Salesforce is an exmaple of SaaS


Advantages of SaaS include:


  • High adoption
  • Lower Initial costs
  • Seamless Integration

    • SaaS vendor clients are all on the same infrastructure and code base, vendors can innovate more quickly and save the valuable development time previously spent on maintaining numerous versions of outdated code.


    SaaS has grown rapidly in recent years, this is down to a number of factors:

  • The growing use of mobile phones
  • Increasing access to broadband
  • Growing popularity of web like interfaces
  • The standardisation of digital technology
